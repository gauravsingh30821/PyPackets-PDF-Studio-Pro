## 🚀 Overview

PyPackets PDF Studio Pro is a modern desktop application designed to simplify PDF management. It provides powerful PDF tools through an easy-to-use graphical interface built with Python and CustomTkinter.

Whether you need to merge documents, split pages, protect files, or convert images into PDFs, PyPackets PDF Studio Pro offers a professional solution.

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/PyPackets-PDF-Studio-Pro.git

cd PyPackets-PDF-Studio-Pro
```

### Install Dependencies

```bash
pip install customtkinter
pip install pypdf
pip install pillow
```

### Run Application

```bash
python PyPackets_PDF_Studio_Pro.py
```

## 🔧 Build EXE

Install PyInstaller:

```bash
pip install pyinstaller
```

Build executable:

```bash
pyinstaller --onefile --windowed --icon=icon.ico PyPackets_PDF_Studio_Pro.py
```

The executable will be generated inside:

```text
dist/
```
